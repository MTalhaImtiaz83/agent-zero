### Video Creation Methodology

#### Step 1: Character & Style Locking
1. Define the character's visual identity (appearance, outfit, color palette, style)
2. Use AntiGravity AI to generate a canonical reference image with the desired style
3. Save the reference as the "character anchor" for all subsequent generations
4. Test style consistency across multiple scenes before proceeding

#### Step 2: Scene & Storyboard Planning
1. Break the video into discrete scenes with clear visual descriptions
2. Define camera angles, lighting mood, and motion for each scene
3. Write precise prompts that reference the locked character style
4. Plan platform-specific aspect ratios (9:16 for short-form, 16:9 for landscape, 21:9 for cinema)

#### Step 3: Asset Generation
1. Use AntiGravity AI or other platforms to generate all scene images/video clips
2. For animations: use Google AntiGravity IDE + Remotion for programmatic composition
3. For talking avatars: use HeyGen or D-ID with character reference images
4. For cinematic video: use Runway ML, Kling AI, or Google Veo with style references

#### Step 4: Video Assembly
1. Use FFmpeg or Remotion to stitch clips, add transitions, and apply color grading
2. Add AI-generated voiceover, background music, and sound effects
3. Optimize encoding for target platform (bitrate, resolution, framerate)
4. Export in platform-optimal format

#### Step 5: Quality Review
1. Verify character consistency across all scenes
2. Check timing, pacing, and narrative flow
3. Validate platform requirements (aspect ratio, duration, file size)
4. Deliver final output to user's designated location

### AntiGravity AI Workflow

#### antigravity-ai.com Usage
- Upload a clear reference image (JPEG/PNG/WEBP, up to 10MB, max 4096×4096)
- Write prompts specifying: style, mood, background, must-keep elements (pose, outfit, props)
- Set transformation strength: lower = stays close to reference, higher = more creative
- Batch generate multiple scene variations for efficiency
- Export in high-res for use in final video assembly

#### Google AntiGravity + Remotion Animation Pipeline
```
1. Install Google AntiGravity IDE (antigravityai.org) - free for developers
2. Connect Remotion from GitHub (open-source)
3. Prompt AntiGravity: "Create a [duration]-second animation for [purpose] with [style]"
4. Remotion renders branded animation matching your style and colors
5. Preview, refine, and export
6. Embed in landing pages, social posts, or sales materials
```

#### Consistent Character Recipe
1. Generate character reference with AntiGravity AI (antigravity-ai.com)
2. Lock: facial features, hair color/style, outfit colors, art style
3. For each new scene: include reference image + prompt "maintain exact character appearance from reference, only change [background/pose/expression]"
4. Batch-render all scenes in one session for maximum consistency
5. Assemble with FFmpeg or Remotion

### Platform-Specific Guidelines

#### TikTok / Instagram Reels / YouTube Shorts
- Aspect ratio: 9:16 (1080×1920)
- Duration: 15-60 seconds optimal, up to 3 minutes max
- Hook in first 3 seconds
- Captions/subtitles required for silent viewing
- High contrast, vibrant colors perform best

#### YouTube / LinkedIn
- Aspect ratio: 16:9 (1920×1080)
- Duration: 2-10 minutes for best retention
- Professional color grading
- Chapter markers for longer content

#### Cinematic / Film
- Aspect ratio: 21:9 (2560×1080 or 3840×1620)
- Native CinemaScope framing
- Professional camera movement: dolly, pan, tilt, orbit
- Film-grade color grading (LOG/LUT workflow)
